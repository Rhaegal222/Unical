## Complessità Asintotica

- notazione $O$ ('o' grande)
- notazione s2

$f(n) \in O(g(n))$  se  $\exists  c,n_0 > 0$   t.c.
$\forall  n > n0$  
$0 \le f(n) \le c \cdot g(n)$$

### Alcuni andamenti

- $O(1)$ costante
- $O(\log(n))$ logaritmico
- $O(n)$ lineare
- $O(nlog(n))$
- $O(n^2)$ quadratico
- $O(n^3)$ cubico
- $O(n^{4^+})$ polinomiale
- $O(2^n)$ esponenziale

```cpp
int fib(int n){     // O(1)
  int f, f1 = 0, f2 = 1;  // O(1)
 
   //caso migliore O(1)
   //caso peggiore O(n)
  if( n==0 || n == 1) return n;  
  else
    for (int i = 0; i < n; i++)
    {   
      f = f1 + f2; // O(1) n volte
      f2 = f1;  //
      f1 = f;   // 1*n --> O(n) 
    }
  return f; // O(1)
}
```
